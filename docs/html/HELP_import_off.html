import_off

Below is a demonstration of the features of the import_off function

Contents

  • Import OFF file as patch data
clear; close all; clc;

Plot settings

faceAlpha=1;
fontSize=10;

Import OFF file as patch data

%Set main folder
defaultFolder = fileparts(fileparts(mfilename('fullpath')));
pathName=fullfile(defaultFolder,'data','OFF');
testCase=1;
switch testCase
    case 1
        offName='elephant-50kv.off';
end
fileName=fullfile(pathName,offName);

The import_off function imports http://www.geomview.org type .off files specifying a surface mesh (provided all faces are of the same type!). The output consists of (patch data) faces F and vertices V.

[F,V] = import_off(fileName);

Plotting the models

cFigure;
title('Imported patch data from OFF file','fontSize',fontSize);

gpatch(F,V,'gw');

axisGeom;
camlight('headlight');
drawnow;
